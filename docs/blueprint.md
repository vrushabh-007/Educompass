# **App Name**: CollegePath

## Core Features:

- User Authentication: User registration and login system with admin and student roles.
- College Matching Dashboard: Dynamic dashboard displaying matched colleges with filtering and search capabilities based on academic scores, exams, preferred country, and financial status.
- AI College Recommendation: AI-powered college recommendation tool based on student scores and preferences. An LLM tool will decide when and if a college should be listed, based on the students profile.

## Style Guidelines:

- Primary color: Blue (#3498db) to convey trust and reliability.
- Secondary color: Light gray (#ecf0f1) for backgrounds and subtle elements.
- Accent: Green (#2ecc71) for positive actions, success indicators, and highlights.
- Clean and readable sans-serif fonts for a modern and professional look.
- Consistent and professional icon set for navigation and data representation.
- Responsive and user-friendly layout optimized for various screen sizes using a grid system.
- Subtle transitions and animations to enhance user experience without being distracting.
